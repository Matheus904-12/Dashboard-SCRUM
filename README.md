# SCRUM Nexus Dashboard

Este é um dashboard premium em tempo real desenvolvido para Product Owners e gestores SCRUM que precisam cruzar dados de **Vendas, Estoque e Logística** enquanto gerenciam sua equipe e demandas.

## 🚀 Funcionalidades

- **Monitoramento em Tempo Real**: Gráficos dinâmicos de vendas e logística.
- **Gestão de Equipe**: Registro de membros do time (PO, Gerente, Devs).
- **Backlog de Demandas**: Registro rápido de demandas de clientes com níveis de prioridade.
- **Design Cyber-Glass**: Interface moderna com glassmorphism, modo escuro e animações fluidas.
- **Leve e Rápido**: Utiliza `localStorage` para persistência, sem necessidade de banco de dados externo complexo.

## 🛠️ Tecnologias

- HTML5 Semântico
- CSS3 (Vanilla com Glassmorphism)
- JavaScript (Vanilla ES6+)
- [Chart.js](https://www.chartjs.org/) para visualização de dados

## 📦 Como usar localmente

1. Clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Comece a registrar sua equipe e demandas!

## 🌐 Deploy no GitHub Pages

Para publicar este dashboard:

1. Vá nas **Settings** (Configurações) do seu repositório no GitHub.
2. No menu lateral, clique em **Pages**.
3. Em **Build and deployment**, selecione a branch `main` e a pasta `/(root)`.
4. Clique em **Save**.
5. O link será gerado em poucos minutos.
